# 前途加科技（Qantara）官方网站

## 项目简介

这是前途加科技（Qantara）的官方网站，展示公司介绍、服务范围、专家资源、课程服务和企业服务等信息。

## 公司信息

- **公司中文名**：前途加科技
- **公司英文名**：Qantara（谐音"前途" + 含义为"桥梁、连接"）
- **Logo设计**：采用科技感圆弧、开放式结构，寓意"连接·成长·前途可期"

## 文件结构

```
qantara-website/
```

## 使用方法

1. **直接打开**
   - 双击 `index.html` 文件即可在浏览器中打开

2. **使用本地服务器（推荐）**
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   
   # 然后访问 http://localhost:8000
   ```

## 自定义配置

### 修改颜色主题
在 `styles.css` 文件的 `:root` 变量中修改：
```css
:root {
    --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    --secondary-gradient: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
    /* ... */
}
```

### 修改联系信息
在 `index.html` 文件的页脚部分修改联系方式。

### 添加实际表单提交功能
在 `script.js` 文件的表单提交处理部分添加实际的API调用。
